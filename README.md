# FarmRentals 🚜

A full-stack web application that connects farmers with local machinery owners to streamline agricultural equipment rentals.

🔗 **Live Demo:** [View Live Demo](https://farmrentals.onrender.com/)

---

## 🛠 Tech Stack

**Frontend:** HTML5, CSS3, JavaScript (Vanilla ES6+)  
**Backend:** Node.js, Express.js  
**Database:** MySQL (Cloud-hosted via Railway)  
**Cloud & DevOps:** Render (Deployment), Cloudinary (Asset Storage)  
**Security:** JWT (JSON Web Tokens), Bcrypt (Password Hashing)

---

## 🌟 Key Engineering Highlights

- **Full-Stack Deployment:** Managed complete lifecycle from local development to production on Render  
- **Cloud-Native Architecture:** Integrated external database hosting and CDN-based image handling  
- **Secure Authentication:** Implemented strong user identity verification and session management  
- **Relational Data Modeling:** Designed schema for users, equipment inventory, and rental workflows
